# Security Policy

## Supported Versions
We actively patch and support the following versions of this project:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.5.x   | :white_check_mark: |
| < 1.5   | :x:                |

## Reporting a Vulnerability
Please do not open public GitHub issues for security vulnerabilities. 

To report a vulnerability, please email us directly at: **srikantsr@gmail.com**

### Please include:
* A detailed description of the vulnerability.
* Clear steps to reproduce the issue (PoC code or scripts).
* Potential impact or attack vectors.

### Our Process
* **Acknowledgment:** You will receive a reply within 48 hours.
* **Triage:** We will validate the bug and assess its severity.
* **Fix & Disclosure:** We aim to patch the issue and publish an advisory within 30 days.
